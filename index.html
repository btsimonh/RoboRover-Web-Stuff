MK802ii based RoboRover with PS2 Eyetoy camera.


MK802ii 'Android' stick running Gentoo linux.
PIC18F4550 on (http://www.tiertex.com/index.php?route=product/product&path=59&product_id=52).
Based on and heavily influenced by RaspberryTank (see raspberrytank.ianrenton.com). Thankyou Ian.

Wowee 'RoboRover' chassis, using original motor driver and IR sensing.
Low level interfacing using modified UBW (http://www.schmalzhaus.com/UBW/), thank you Brian.
The UBW modifications include IR transmit and receive routines acting on multiple ports, giving sensing information which will be used for autonomous running.
High level code compiled and run on the mk802ii; RaspBerry tank inspired.
HTML serving and PHP via Lighttpd.
2 x 1800Mah lipo (from DVD players?) give a good running time.
Battery sense present; next to do is charger hunting.
Pictures by Sony (PS2 Eyetoy camera).
All interfacing through port 80....

Control interface:

Bars to top/left/right/bottom of video represent IR sense.
Yellow -> far contact
Red -> near contact.
IR front is pulsed with high and low power to get range sens options.

Right sensor is out at the moment.

Autonomy is not yet implemented.
Control Images are not right.
Be gentle.

Video is served via Mjpeg-Streamer to port 8080. But this is accessed through a PHP page which reads the data, and forwards it on to the requester.
Control is done through port 3000, again forwarded by PHP.
